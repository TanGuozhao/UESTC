# UESTC 学习资料与通知归档

本仓库用于归档电子科技大学相关学习资料、政策通知、实训材料、竞赛文件、境外交流项目资料及常用考试材料，方便后续检索、查阅和备份。

内容仅作个人学习与资料整理用途，完全免费，禁止据此盈利。

> 说明：政策通知、评审细则、竞赛规则、报名安排、交流项目要求等内容可能随年份、学院安排和主办方要求调整，请始终以学校、学院及活动主办方最新正式通知为准。

## 仓库结构

仓库目前按主题进行分类整理，常见目录包括：

- `书籍/`：课程参考书、教材与学习资料
- `保研/`：推免通知、成绩计算细则、科创加分、期刊会议参考目录等
- `出国/`：境外交流、短期访学、暑期项目通知、附件与说明材料
- `实训/`：暑期实训、生产实习、企业实训、学分认定与申请表
- `就业/`：就业相关资料
- `竞赛/`：各类竞赛通知、规则、附件、名单与项目材料
- `综测/`：综合素质测评、奖学金、发展性素质加分等文件
- `考研/`：考研相关资料
- `考证考试/`：计算机等级考试等考证资料
- `课程/`：课程相关资料与补充文件

整体整理原则是：

- 先按主题分类
- 再按年份、项目或赛事分目录
- 能单独成组的项目尽量补 `README.md`
- 文件名尽量保留原始通知中的年份、届次、项目名或附件编号

## 目录说明

### 书籍

主要收录基础课程与专业课程资料，如高等数学、线性代数、大学物理、深度学习、计算机组成原理、软件工程等方向的教材与参考书。

### 保研

收录推免相关通知、智育成绩计算细则、科创成果加分细则、核心期刊目录、CCF 推荐会议与期刊目录等资料，便于集中查阅。

### 出国

收录学校及学院相关的境外交流、短期访学、暑期项目资料。对于材料较完整的项目，会在目录内补充单独的 `README.md`，方便快速查看报名条件、时间安排、材料要求和附件用途。

### 实训

收录信息与软件工程学院暑期实训、企业实训、学院工程训练班、短期出国（境）交流项目、大创项目、生产实习学分转换等材料。部分年度目录内已经补充说明型 `README.md`，便于快速定位报名和认定要求。

### 竞赛

收录各类大学生竞赛资料，包括但不限于软件类、人工智能类、创新创业类、数学类、职业规划类、物理实验类等通知、规则、附件、获奖名单与项目材料。

### 综测

收录综合素质测评、国家奖学金、优秀学生奖学金、专项奖学金、发展性素质加分等文件。

### 考证考试

收录全国计算机等级考试、C 语言考纲、数据库技术样题及参考答案等资料。

## 推荐使用方式

可以直接在 GitHub 页面中使用文件搜索，也可以在本地用 PowerShell 快速定位资料：

```powershell
# 按关键词查找文件名
Get-ChildItem -Recurse -File | Where-Object { $_.Name -like "*保研*" }

# 查找所有 PDF
Get-ChildItem -Recurse -File -Filter *.pdf

# 查看某个目录下的全部文件
Get-ChildItem .\竞赛 -Recurse -File
```

如果你想按时间查资料，可以结合仓库根目录的 `timeline.md` 一起使用。

## 维护建议

新增资料时建议尽量保持统一命名，例如：

```text
主题目录/年份-项目名/文件
主题目录/年份-通知标题.扩展名
```

例如：

```text
出国/2026-启梦留学计划/附件2-申请表.doc
竞赛/2026-中国大学生服务外包创新创业大赛/区域赛通知.pdf
实训/2026-软院年实训相关文件/附件1：实训报告模板.docx
```

如果某个项目资料较多，建议在项目目录下增加一个简短 `README.md`，说明：

- 项目简介
- 关键时间点
- 报名条件
- 材料清单
- 目录内各附件用途
