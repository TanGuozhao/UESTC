本仓库用于归档电子科技大学相关学习资料、政策通知、实训材料、竞赛文件和常用考试资料，方便后续检索、查阅和备份。内容仅作辅助学习用，且完全免费，禁止凭此盈利。

> 说明：本仓库仅作个人学习与资料归档用途。政策通知、评审细则、竞赛规则等内容可能随年份和学院要求调整，请以学校、学院及竞赛主办方最新正式通知为准。

## 推荐查找方式

可以直接在 GitHub 页面使用快捷键 `t` 搜索文件名，也可以本地使用 PowerShell：

```powershell
# 按关键词查找文件名
Get-ChildItem -Recurse -File | Where-Object { $_.Name -like "*保研*" }

# 查找所有 PDF
Get-ChildItem -Recurse -File -Filter *.pdf

# 查看各目录文件数量
Get-ChildItem -Directory | ForEach-Object {
  $count = (Get-ChildItem $_.FullName -Recurse -File).Count
  "{0}: {1}" -f $_.Name, $count
}
```

## 目录说明

### 书籍

包含基础课程与专业课程参考资料，当前主要覆盖高等数学、线性代数、大学物理、深度学习、计算机组成原理、软件工程等方向。

### 保研

包含推免工作通知、智育成绩计算细则、科创成果加分细则，以及北大中文核心、CCF 推荐会议/期刊、通信/计算机常见 A 类会议等参考目录。

### 实训

包含信息与软件工程学院暑期实训、企业实训、学院工程训练班、短期出国交流、大创项目、生产实习学分转换等材料。

### 竞赛

包含多类大学生竞赛通知、附件、规则和题目资料，例如：

- “揭榜挂帅”擂台赛
- “中国软件杯”大学生软件设计大赛
- 全国大学生计算机设计大赛
- 全国大学生软件创新大赛
- 全国大学生物理实验竞赛
- 全国大学生机器人大赛 ROBOCON
- 大学生网络文化艺术节
- 市场调查大赛
- 数学竞赛
- 职业规划大赛

### 综测

包含信息与软件工程学院综合素质测评、国家奖学金、优秀学生/专项奖学金、发展性素质加分等文件。

### 考证考试

包含全国计算机等级考试、C 语言考纲、三级数据库技术样题及参考答案等资料。

## 维护建议

新增资料时建议遵循：

```text
主题目录/年份-简短标题.扩展名
```

例如：

```text
保研/2027-信息与软件工程学院推免工作通知.pdf
竞赛/全国大学生计算机设计大赛/2026大赛预通知.pdf
实训/2026-软院年实训相关文件/附件1：实训报告模板.docx
```

建议保留原始通知文件名中的年份、学院、竞赛名称和附件编号，方便后续检索。
