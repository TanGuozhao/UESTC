# 第二十六届启梦留学计划

本目录用于归档第二十六届“启梦留学计划”相关通知、项目介绍与报名材料，便于统一查看、下载和整理提交材料。

## 通知概览

学校面向家庭经济困难且品学兼优的本科学生启动第二十六届“启梦留学计划”选拔工作，旨在帮助学生开拓国际视野、提升综合素养与跨文化交流能力，助力成长为具有国际视野的复合型人才。

根据学院安排，本院拟选拔 `1 名` 家庭经济困难学生推荐到学校，参加 `2026 年暑期赴香港城市大学“数智启航 迈向未来”暑期项目`。

## 项目介绍

### 项目概要

- 时间：`2026年7月26日-8月1日`（7天）
- 地点：香港城市大学
- 主题：`数智启航 迈向未来`
- 授课语言：英文及/或中文

课程由香港城市大学知名教授授课，主要涵盖以下方向：

- 人工智能（AI）
- 计算与数据科学
- 环球市场
- 创新创业
- 相关交叉领域

### 成果收获

圆满完成项目的学生将获得由国际处与教务处共同核发的 `创新与素质拓展学分`。

### 资助方式

入选学生将获得学校出国留学专项经费 `全额资助`。

资助一般包含：

- 项目费
- 往返机票
- 在外住宿费
- 境外保险费
- 集体活动期间地面交通费

以下费用通常不包含在资助范围内：

- 证件办理费
- 签证费
- 在外期间餐费
- 私人消费
- 其他相关个人支出

## 报名与选拔

### 报名条件

申请人需满足以下基本条件：

- 我院家庭经济困难、学业成绩优秀的在读全日制普通本科学生
- 此前未享受过学生出国（境）留学短期奖助学金
- 此前未获得其他同类型出国（境）留学奖助学金的访学经费资助

### 选拔方式

本次选拔按照“学校统筹、学院负责、公平公正、择优推荐”的原则进行。学院将依据学校分配名额开展推荐，选拔过程规范，结果公开透明。

温馨提示：

- 通过选拔并完成公示的学生，需尽快预约并办理 `港澳通行证`
