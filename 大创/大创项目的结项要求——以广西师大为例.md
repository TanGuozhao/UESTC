# 大创项目的结项要求——以广西师大为例

很多同学一听到“大创结项”，第一反应就是：`是不是必须发论文？`

其实不是。

如果按广西师大的这份管理文件来看，不管是`自治区级项目`还是`国家级项目`，结项都不是只有“论文”这一条路，而是`至少满足其中一类成果要求`即可。当然，项目成果必须和立项内容相符，而且项目成员要实际参与，不能随便拿一个不相关的成果来凑。

这篇就尽量用好理解的话，把大创结项要求梳理一遍。

## 一、先说最核心的一句话

大创结项不是只看你“做没做完”，而是看你`有没有形成符合要求的成果`。

以广西师大为例：

1. `自治区级项目`：至少达到规定条件中的`一项`
2. `国家级项目`：至少达到规定条件中的`一项`

也就是说，结项看的是“成果达标没有”，不是单纯看你写了多少周报、开了多少会。

## 二、自治区级项目怎么结项？

自治区级项目结项，至少满足下面几类成果中的一类。

### 1. 论文结项

如果你打算用`论文`结项，要求是：

1. 至少提交`1篇`与项目密切相关的学术论文
2. 论文需要在`国内公开发行的学术期刊`上发表
3. 论文要能在`中国知网、万方数据库、维普`三大平台查询到
4. 项目成员需要是`第一作者`，或者由`指导老师第一作者、项目负责人第二作者`
5. 论文中要`标注项目资助信息`

通俗一点说，就是这篇论文不能只是“沾边”，而要和你的大创项目直接相关；同时作者署名和资助标注也要符合要求。

### 2. 知识产权结项

如果你打算用`知识产权`结项，要求是：

1. 至少提交`1项`与项目相关的专利申请受理通知书
2. 这里包括`实用新型专利`或`外观设计专利`
3. 或者提交`软件著作权登记证书`

这条对做系统、做软件、做产品设计的项目会比较友好。

### 3. 竞赛获奖结项

如果你打算用`比赛获奖`结项，要求是：

1. 必须是《广西师范大学全日制普通本科生学科竞赛管理办法（2023年修订）》中所列的竞赛项目
2. 在`自治区级及以上`竞赛中获得`三等奖及以上`
3. `项目负责人`要是参赛团队负责人
4. 需要提供`获奖证书`

这意味着，不是所有比赛都能拿来结项，关键要看这个比赛是否在学校认定范围内。

### 4. 实物成果结项

如果你是做`软件、系统、设计作品、图纸、音乐、影视、美术、实物作品`这一类，也可以走实物成果结项。

文件的表述是：由专家根据以下因素综合判断是否满足结项条件：

1. 创新程度
2. 难易程度
3. 影响力
4. 其他成果情况

这类结项没有前几类那么“硬指标化”，但也不是说随便做个成品就一定能过，最后还是要看专家评审。

### 5. 注册公司结项

如果项目已经走到公司化这一步，也可以用`注册公司`作为结项依据，但条件比较明确：

1. 项目负责人在公司股份占比`不低于10%`
2. 至少提供`3名`本校在校学生或毕业生的工作岗位
3. 公司运营至少`6个月`
4. 年均营业额达到`5万元以上`
5. 需要提供`企业运营情况报告`
6. 如果是靠融资作为支撑，需要获得投资人或投资机构`5万元以上`投资，并提供`投资合同`

这条门槛相对高一些，更适合已经实质性创业的项目。

## 三、国家级项目怎么结项？

国家级项目的逻辑和自治区级差不多，也是在几类成果里`至少满足一类`，但整体要求会更高。

### 1. 论文结项

国家级项目如果走论文结项，要求是：

1. 至少提交`2篇`与项目相关的学术论文
2. 论文需在`省部级及以上期刊`发表
3. 或者在`核心期刊`发表与项目相关论文`1篇`
4. 项目成员须为`第一作者`，或者`指导老师第一作者、项目成员第二作者`
5. 同样要`标注项目资助信息`

简单理解：国家级项目对论文数量和质量要求都比自治区级更高。

### 2. 知识产权结项

国家级项目如果走知识产权结项，要求是：

1. 至少提交`2项`与项目相关的专利申请受理通知书
2. 包括`实用新型专利`或`外观设计专利`
3. 或者提交`软件著作权登记证书`

和自治区级相比，最明显的差别就是：`数量要求更高`。

### 3. 竞赛获奖结项

国家级项目如果走竞赛成果结项，要求是：

1. 竞赛必须属于《广西师范大学全日制普通本科生学科竞赛管理办法（2023年修订）》中的项目
2. 获得`自治区级竞赛二等奖及以上`奖项
3. `项目负责人`要是参赛团队负责人
4. 需要提供`获奖证书`

这里和自治区级项目的一个明显区别是：自治区级项目是`自治区级及以上三等奖及以上`，国家级项目则提高到`自治区级二等奖及以上`。

### 4. 实物成果结项

国家级项目也可以用`自主研发的软件、系统、设计说明书、图纸、音乐、影视、美术、实物作品等`作为主要结项依据。

同样是由专家根据以下因素综合判断：

1. 创新程度
2. 难易程度
3. 影响力
4. 其他成果情况

也就是说，这一类没有完全量化成“必须几篇论文、几项专利”，但评审时一般会更看重成果质量和完成度。

### 5. 注册公司结项

国家级项目如果走公司化结项，要求比自治区级更高：

1. 项目负责人在公司股份占比`不低于10%`
2. 至少提供`5名`本校在校学生或毕业生的工作岗位
3. 公司运营至少`12个月`
4. 年均营业额达到`10万元以上`
5. 年均利润达到`1万元以上`
6. 需要提供`详细的企业运营情况报告`
7. 如果是靠融资作为支撑，需要获得投资人或投资机构`10万元以上`投资，并提供`投资合同`

这一条基本已经不是“做个学生项目”那么简单了，而是比较接近真正创业落地。

## 四、自治区级和国家级，差别主要在哪？

如果只看结项要求，国家级项目整体上就是`更高一档`。

最直观的差别包括：

1. `论文`：自治区级一般是`1篇`，国家级一般是`2篇`，或者核心期刊`1篇`
2. `知识产权`：自治区级一般是`1项`，国家级一般是`2项`
3. `竞赛获奖`：自治区级要求`自治区级及以上三等奖`，国家级要求`自治区级二等奖及以上`
4. `注册公司`：国家级在岗位数、运营时长、营业额、利润、融资额度上都更高

所以如果一个项目本身基础一般、成果积累有限，后期做国家级项目的结项压力通常会明显更大。

## 五、还有一个很容易被忽略的点：成果必须和立项内容一致

文件里专门强调了两件事：

1. `所有成果需与立项内容相符`
2. `项目成员必须参与`

这句话很重要。

意思就是：

1. 不能拿一个和项目关系不大的论文、比赛、专利来硬凑结项
2. 也不能成果全靠别人完成，项目成员自己没实际参与

结项不是“交材料比赛”，而是要能说明：`你这个项目真的做过，而且成果确实是围绕这个项目产出的。`

## 六、成果发表或申请时，要记得标注资助信息

这一点也很关键，尤其是发论文、申报成果的时候容易忘。

文件要求：项目所有成果在发表或申请时，要标注对应的大创项目资助信息，也就是类似下面这种格式：

`自治区级（或国家级）广西师范大学大学生创新创业训练计划项目资助——xxx（资助编号）`

如果是英文标注，文件也给了参考格式。

### 1. 自治区级项目英文标注参考

`Project xxx (Grant Number: ****) supported by Guangxi Normal University’s Provincial Training Program of Innovation and Entrepreneurship for Undergraduates`

### 2. 国家级项目英文标注参考

`Project xxx (Grant Number: ****) supported by Guangxi Normal University’s National Training Program of Innovation and Entrepreneurship for Undergraduates`

其中：

1. `xxx`代表项目具体名称
2. `****`代表资助编号

这个细节千万不要小看。有些成果本身够了，但如果资助标注不规范，后面补材料会很麻烦。

## 七、最后给一个最实用的理解

如果你现在正在做大创，结项时可以先问自己三个问题：

1. 我的成果到底属于`论文、知识产权、竞赛、实物、公司`中的哪一类？
2. 我的成果是否`和立项内容直接相关`？
3. 我的成果是否已经达到`自治区级`或`国家级`对应门槛？

如果这三个问题都能回答清楚，结项准备基本就不会太乱。

## 八、一句话总结

大创结项不是只有“发论文”这一条路，但不管走哪条路，核心都是：`成果真实、成果相关、成果达标、资助标注规范。`
