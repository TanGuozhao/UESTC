# 2026年本科暑期实训通知

本目录用于整理 2026 年信息与软件工程学院本科暑期实训相关通知、附件与学分认定材料，方便统一查阅和后续提交。

## 实训时间及安排

具体安排请以学院官网通知为准：

- 官网链接：`sise.uestc.edu.cn/info/1033/14405.htm`

建议同学们优先查看官网原通知，确认时间安排、学分认定要求和后续补充说明。

## 特别注意

### 1. 企业实训

请重点参考以下附件材料：

- `企业实训岗位需求汇总2026`
- `2026“萃青于成·筑梦同行”驻区高校大学生暑期实践计划`

注意事项如下：

- 应企业要求，为避免岗位空缺，简历不要多投。
- 如选择服从调剂，则有调剂机会。
- 安恒、成华区岗位仅筛选简历，无需面试。
- 只有在确定要去的前提下再投递对应岗位。
- 除特别说明岗位外，其他企业简历统一投递至：
  - `https://send2me.cn/TcZazhj9/QfOeRw6Q3pes8Q`
- 企业后续会自行联系同学安排面试。
- 简历投递截止时间：`2026年6月1日 12:00`

### 2. 企业暑期实训班、学院工程训练班

该部分通知会在“企业实训”基本确定后另行发布，请留意学院后续通知。

### 4. 短期出国（境）交流项目

可参考已发布的交流项目通知，并按相关要求进行认定。

### 5. 大创项目

## 学分认定特别提醒

通过以下途径进行暑期实训学分认定的同学：

- 学科竞赛
- 短期出国（境）交流项目
- 大创项目

必须在 `10月上旬前` 获得相关奖状、证书或证明材料，否则可能影响最终认定。

## 使用建议

- 先确定自己准备走哪一类认定路径，再有针对性查看对应附件。
- 企业实训同学优先准备简历，并注意不要重复投递多个岗位。
- 凡涉及报名、面试、认定、证明材料的时间节点，务必以学院官网和辅导员通知为准。
