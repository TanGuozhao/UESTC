## 评分标准

注：课外锻炼和健康跑按照实际情况由系统计分。课外锻炼按 `锻炼次数 / 36 × 10` 四舍五入计分；健康跑按 `1 公里 = 0.2 分` 计分，不足 1 公里不计分。

### 体育竞赛及活动评分表

| 类别 | 子类别 | 得分（总分 10 分） |
| --- | --- | --- |
| A 省级竞赛及活动 | A1 参加省级及以上体育竞赛及活动 | 10 分 |
| B 校级竞赛及活动 | B1 校级竞赛及活动获得名次或奖励 | 每项 5 分 |
| B 校级竞赛及活动 | B2 参与校级竞赛 | 每项 2 分 |
| B 校级竞赛及活动 | B3 参与最强班级联赛、经批准的院级选拔赛 | 每项 1 分，最高 5 分 |
| B 校级竞赛及活动 | B4 参与校级体育活动、体育部举办和认定的体育讲座、赛事志愿者 | 每项 1 分 |
| C 体质测试 | C1 本年度体质测试总成绩达到 80 分及以上 | 5 分 |
| C 体质测试 | C2 本年度体质测试总成绩达到 70 分-79.99 分 | 4 分 |
| C 体质测试 | C3 本年度体质测试总成绩达到 60 分-69.99 分 | 3 分 |
| D 线上课程 | D1 完成学校线上体育课程的学习和考试 | 每门 5 分 |

注：不同比赛和活动可以累计，单项联赛最高得 5 分，总分不能超过 10 分。同一竞赛及活动的得分项不能累计，只计最高得分（不同组别比赛不能累计，名次和参与得分不能累计）。
